# KalaAwakasha Web App

Mobile-first organizer web app using React + Vite + TypeScript + Tailwind.

## Features

- Login with Supabase Auth
- Dashboard counts
- Add ticket with optional photo upload
- Bulk CSV import
- Ticket search and details
- QR scan and manual verify
- Check-in confirmation
- Check-in history with realtime updates

## Setup

1. Copy `.env.example` to `.env`.
2. Fill `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
3. Install and run:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
