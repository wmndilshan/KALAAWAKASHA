# KalaAwakasha Mobile App

Expo + React Native organizer app optimized for gate operations.

## Features

- Login with Supabase Auth
- Fast QR scanner flow
- Immediate valid/invalid/duplicate states
- Student photo preview
- One-tap check-in
- Manual verify fallback
- Search ticket and ticket details
- Recent activity logs

## Setup

1. Copy `.env.example` to `.env`.
2. Fill `EXPO_PUBLIC_SUPABASE_URL` and `EXPO_PUBLIC_SUPABASE_KEY`.
3. Install and run:

```bash
npm install
npm start
```

Then press `a` for Android emulator/device or `i` for iOS simulator.
